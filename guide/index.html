    # 入门

    欢迎使用 Goof off!

    # Goof off是什么？

    Goof off是一款让你偷闲摸鱼的软件(暂时只支持window平台，不支持Mac和linux)，内置小说，网页游戏，视频直播和抖音短视频等页面工具，你可以轻松设置背景透明和窗口大小，隐秘式的在工作页面内轻松摸鱼，另外还支持自定义网页内容，输入任意网页即可隐蔽浏览，同时还加入了老板快捷键，Ctrl+B直接隐藏窗口，不用担心被人发现摸鱼了~

    # Goof off有什么好处？

    • 平时工作忙完了，感觉时间过得太慢，开个窗口，调低透明度，看个直播，刷个短视频，来几局麻将，刷会原神，不知不觉就到了下班的点~
    • 浏览器页面浏览网页太麻烦，窗口太大，隐蔽性不好，直接在Goof off输入您想看的链接，页面置顶，随意浏览

    注：本软件只提供让您忙里偷闲的方式，不额外承担您摸鱼被开除的风险责任~
