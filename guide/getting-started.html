# 功能介绍

# 小说

小说支持在线阅读，多个书源，在线搜索您想看的，阅读界面直角尺调整字体大小，字体颜色，背景透明或背景颜色，通过拉伸窗口大小，将窗口定格在方便阅读的位置，置顶页面，随意阅读。

支持调整快捷键，自定义上一章下一章快捷键，显示隐藏工具栏等功能

页面展示

# 在线原神

让你能在网页上玩原神

页面展示

# 红白机、麻将功能

打开红白机或麻将功能，调低透明度和窗口大小，即可随意畅玩

# 短视频、在线视频等功能

视频功能支持视频全屏，无论是刷抖音，刷b站还是看剧，找到您想看的视频，调整透明度和窗口大小，一键视频全屏去除其它页面干扰，在别人看不到的角落里安心的看视频~

# chatGpt

Goof off将维护一个chatGpt入口，让您能够一直免费使用chatGpt~

# 最右

通过js脚本注入，让您的电脑变成一个移动端，随时随地刷最右

# 脚本注入

页面支持自定义脚本注入，您可以将您喜欢的脚本注入到页面中，让页面变得更好看，更实用

建议使用之前在对应的页面进行测试，避免脚本注入错误导致页面无法使用，不建议不熟悉js脚本的用户进行尝试~

# 网页自定义

除了以上的这些功能外，Goof off还支持自定义您想看的网页，输入网页名称和链接，一步即可添加网页

# 快捷键说明

ctrl+Q :切换首页隐蔽模式,在隐蔽模式下，图标将进行缩小，隐藏功能名称，调低透明度为0.1，隐藏除功能外的元素

ctrl+B :老板键，点击将切换显示隐藏界面，隐藏界面时将软件隐藏到托盘下，可自定义

ctrl+Z :小说界面，上一章快捷键，可自定义

ctrl+X :小说界面，下一章快捷键，可自定义

ctrl+h :小说界面，隐藏和显示设置栏，可自定义
